## How to run this APP

1. Clone this project
```
git clone https://github.com/wawat7/wings.git
```

2. Go to folder project
3. Create file .env to setup database
4. Run Migration
```
php artisan migrate
```
5. Run Seeder to initiate data user and products
```
php artisan db:seed
```
6. Run Application
```
php artisan serve
```
